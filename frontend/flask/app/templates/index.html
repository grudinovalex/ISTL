Welcome!

Interactive SysAdmin Training Lab (ISTL) is a Linux training platform that aims to turn you into an experienced System Administrator.

You will have a chance to complete a number of exercises.

These exercises are grouped into chapters, each chapter explores a single Linux topic.

You have your Linux Machine which can accessed by clicking "Linux Machine" on the nav-bar.

All the exercises need to completed on the Linux Machine!

Once you are finished, click the check button to proceed.

The exercise files related to each chapter are stored in your home folder.



If you accidentally damage/delete the exercise files, go to the "Restore" page and restore the affected chapter to factory settings.

If you have major issues with the server that you are unable to fix, open the terminal on your local machine, go to the interactive-sysadmin-training-lab directory (which you cloned from github / gitlab) and run the restart script by executing the following command:

./restart

Once you have completed all the exercises, you can shut down ISTL by running the stop script, execute the following command on you local machine (while in the cloned ISTL git repository):

./stop {% endblock %}
